# UNEC Secure Note System

A Python-based secure note management application with user authentication and encryption.

## Features
- User registration and login system
- SHA-256 password hashing
- Fernet symmetric encryption
- Password complexity validation
- Limited login attempts
- File-based secure storage

## Technologies
- Python
- Cryptography (Fernet)
- Hashlib
- Regex
- File Handling

## Security Concepts Applied
- Authentication
- Encryption
- Hashing
- Access Control
- Secure Coding Principles
